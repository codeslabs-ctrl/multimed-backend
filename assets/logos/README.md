# Logos para PDF (backend)

Las rutas guardadas en BD (p. ej. `logo_path`, `logo_path_recipe`) como `/assets/logos/mi-logo.png` se leen **desde esta carpeta del API** (`demomed-backend/assets/logos/`), no desde el `src/assets` del frontend Angular.

Coloca aquí los PNG/WebP/SVG que uses en informes y recetas, y vuelve a generar el PDF.
